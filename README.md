# FinançasEmOrdem

Projeto de controle financeiro para organizar e gerenciar suas finanças de maneira simples e eficiente.

## Funcionalidades

- Cadastro de transações financeiras
- Relatórios de despesas e receitas
- Gráficos para visualização do fluxo financeiro
- Configurações personalizadas para categorias de despesas

## Tecnologias Utilizadas

- Backend: Node.js, Express
- Frontend: React 
- Banco de Dados: PostgreSQL
